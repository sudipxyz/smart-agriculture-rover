Smart Agriculture Rover

Automated Field Monitoring and Irrigation System

Project Overview

This rover is designed for precision agriculture. It monitors environmental and soil conditions, avoids obstacles, and automatically irrigates the soil when moisture levels are low. The entire system is solar-powered and controlled wirelessly via web servers hosted on ESP32 and ESP8266 microcontrollers.

Microcontrollers

  • ESP32: Main brain of the rover. It controls motor movement, reads from MPU6050 for balance, processes GPS location, detects obstacles using ultrasonic and IR sensors, and hosts the main web interface.
  • ESP8266 (Sensor Node): Collects environmental data (humidity, temperature, air pressure, rain, and soil moisture) and transmits it to the control node wirelessly.
  • ESP8266 (Control Node): Receives soil data wirelessly and controls a water pump accordingly. Sends alerts about storm, rain, or humidity rise to farmers via the interface.

Sensors and Modules Details

  • GPS Module: Provides real-time location of the rover for tracking and navigation.
  • MPU6050: A 6-axis gyroscope and accelerometer used to maintain balance and orientation of the rover.
  • Ultrasonic Sensor: Measures distance to obstacles in front to prevent collisions. Uses sound waves.
  • IR Sensor: Detects obstacles behind the rover by sensing reflected infrared light.
  • DHT11 Sensor: Measures ambient temperature and humidity. Used to monitor weather conditions.
  • BMP180 Sensor: Measures barometric pressure. Helps in predicting weather changes like storms.
  • Rain Sensor: Detects presence of rain. Triggers alerts to farmers.
  • Soil Moisture Sensor: Detects water content in the soil. Triggers irrigation if moisture is below threshold.

How It Works

  1. The ESP32 controls the rover movement using high torque gear motors and avoids obstacles using ultrasonic (front) and IR (back) sensors.
  2. MPU6050 ensures the rover stays balanced on uneven terrain.
  3. The rover collects environmental data via DHT11, BMP180, and Rain sensor, processed by ESP8266 Sensor Node.
  4. Soil moisture is also checked and transmitted to the ESP8266 Control Node.
  5. If moisture is low, the control node activates the water pump wirelessly.
  6. Web servers on ESP32/ESP8266 show sensor data and allow remote control.
  7. System runs on 12V battery, charged via solar or adapter.

Web Server Access

  1. Connect your mobile/laptop to the Wi-Fi hotspot created by ESP32 or ESP8266.
  2. Open your browser and enter the IP address shown in serial monitor (e.g., 192.168.4.1).
  3. Access live data, control the water pump, and monitor environmental changes in real time.

Power Management

The system runs on a 12V lithium battery. A solar panel is mounted for continuous power. Adapter charging is also available as backup.
